GitHub Repository klonen, bearbeiten und wieder pushen

1. Auf github.com ein neues public Repository erstellen

2. mit dem Befehl git clone (Link des Repository) das Repository klonen

3. Nach wunsch bearbeiten

4. Folgende Befehle eingeben

4.1. git config --global user.mail (E-Mailadresse)

4.2. git config --global user.name (Benutzername)

4.3. git config --global password (Passwort)

5. Visual Studio Code öffnen

6. Ordner mit geklontem Repository öffnen

7. Repository mit Commit Nachricht pushen
